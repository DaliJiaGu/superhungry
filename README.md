# superhungry
一个基于Vue2全家桶开发的点餐系统
项目采用前后端分离的团队模式，本仓库仅限存储本人编写的部分前台页面代码
## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
